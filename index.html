What is Pachino?

Pachino is yet another tool for productivity enhancement.

So what is so new and interesting to brag about?

Pachino is web-based, open source and aims to create a community so that you can have the same fun doing the real stuff as you currently have playing on your favorite social web site.

What is the method?

Pachino is a tool, so you can use it with whatever method it fits. Of course it will rely on well-known principles and existing methodologies.
